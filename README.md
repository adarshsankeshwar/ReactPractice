# 🚀 React Practice Session

This repository contains a complete React practice session covering all the **important core and advanced concepts of React.js**. It is designed for beginners to intermediate learners who want to strengthen their React fundamentals through hands-on coding.

---

## 📌 What This Project Covers

### 1. React Basics
- What is React
- JSX (JavaScript XML)
- Components (Functional Components)
- Props and Prop Drilling
- State Management using `useState`

---

### 2. React Hooks
- `useState` – state handling
- `useEffect` – lifecycle methods replacement
- `useRef` – DOM manipulation & persistent values
- `useMemo` – performance optimization
- `useCallback` – function memoization
- Custom Hooks (basic understanding)

---

### 3. Component Communication
- Parent to Child (Props)
- Child to Parent (Callback functions)
- Sibling communication (via parent)

---

### 4. Conditional Rendering
- if-else rendering
- ternary operators
- logical AND (`&&`) rendering

---

### 5. Lists & Keys
- Rendering arrays using `.map()`
- Importance of unique `key` prop

---

### 6. Forms in React
- Controlled Components
- Form validation basics
- Handling multiple inputs

---

### 7. React Router
- Installation and setup
- Routes, Route, Link
- Dynamic routing
- useNavigate hook

---

### 8. State Management (Basic)
- Context API
- Global state sharing

---

### 9. API Integration
- Fetching data using `fetch`
- Using `axios`
- Handling loading and error states
- useEffect with API calls

---

### 10. Styling in React
- Inline styling
- CSS files
- CSS Modules
- Basic introduction to Tailwind CSS (optional)

---

### 11. Performance Optimization
- React.memo
- useMemo and useCallback usage
- Avoiding unnecessary re-renders
